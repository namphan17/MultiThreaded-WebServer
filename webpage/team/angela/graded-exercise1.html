Questions

  1. Explain why it would be wrong to say that the network layer and the transport do the same job.
    • As opposed to transport layer which is a logical communication between processes running on different hosts the Network-layer protocol provides logical communication between hosts. subtle but important. Note that Transport layer does it all on the ends where as the network layer move the messages throughout the networks (188-9)

  2. Why is pipelining an important concept and what does it allow?
    • Pipelining allows sender to send multiple packets without waiting for acknowledgments and that is important because it allows for fast communication between systems are far away and this how we have the internet as we know it now.(215-19)

  3. What is the difference between Go-Back-N and selective repeat.
    • Go-Back-N (GBN) protocol, the sender is allowed to transmit multiple packets (when available) without waiting for an acknowledgment, but is constrained to have no more than some maximum allowable number,N,of unacknowledged packets in the pipeline. A single packet error can thus cause GBN to retransmit a large number of packets, many unnecessarily. Where as selective-repeat protocols avoid unnecessary retransmissions by having the sender retransmit only those packets that it suspects were received in error (222-224)

  4. Explain in detail explain the difference between the TCP Congestion Control and flow control
    • Flow-control service to its applications to eliminate the possibility of the sender overflowing the receiver's buffer. Flow control is thus a speed-matching service

  5. Describe why using Forwarding and Routing interchangeably is not a good choice in this class
    • In lots of scenarios the two are interchangeably but in the context of this class they are not they have different meaning for Forwarding: refers to the router-local action of transferring a packet from an input link and Routing:refers to the network-wide process that determines the end-to-end paths that packets take from source to destination

  6. What algorithm has three major components: (1) slow start, (2) congestion avoidance, and (3) fast recovery.
    • TCP congestion control

  7. What is the relationship between utilization and pipelining?
    • The utilization determines how much gets through the pipeline

  8. Which layer is all about logical communication and what does that mean?
    • Transport layer Implemented on the end servers but not in the network routers basically this layer sends messages with out the two worrying about physical infrastructure involved.

  9. What and why is explicit forward congestion indication (EFCI) bit important
    • (268)Each data cell holds it. A congested network switch can set the EFCI bit in a data cell to 1 to signal congestion to the destination host.

  10. What is TCP congestion control often referred to and why?
    • TCP congestion control is often referred to as an additive-increase, multiplicative-decrease (AIMD)form of congestion controlTCP's congestion control consists of linear (additive) increase in cwnd of 1 MSS per RTT and then a halving (multiplicative decrease) of cwnd on a triple duplicate-ACK event.
