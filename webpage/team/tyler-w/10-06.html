What is DNS?

The Domain Name System

The domain name system, or DNS, is a heirarchical decentralized naming system for computers, services, or any resource connected to the internet or a private network. It associates various information with domain names assigned to each of the participating entities. Hosts are identified by a host-name and an IP address. This is because it is easy to identify by humans from the host-name and easy to identify by the network from the IP address, and so must be translated accordingly.

The DNS is

  • a distributed database implemented in a heirarchy of DNS servers
  • an application-layer protocol that allows hosts to query the distributed database

DNS is commonly employed by other application-layer protocols including:

  • HTML
  • SMTP
  • FTP
