Link State vs. Distance Vector

Link State Protocol
Link state protocol creates a map of the network and then determines the fastest path through the network using Dijkstra's algorithm, but before the algorithm can be run a map of the network needs to be built. To do this the following steps are taken:

  1. The neighbours of each node, in case the routers, are found
  2. Each router periodically checks its connection with each of its neighbours, and uses this to figure out each connections “weight”
  3. Using the weights, a map of the fastest path from any two routers can be created


Distance Vector Protocol
Each node (router) knows only two things about the network around it:

  1. The direction the packet is going
  2. The distance of that packet from its destination

Using this information each router passes the packet roughly in the direction that it needs to go, using the shortest path.
