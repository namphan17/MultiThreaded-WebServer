Link Layer Vocab

  • Node: Any device that runs through a link-layer protocol
  • Links: Communication channels that connect adjacent nodes
  • Frames: Link Layer packets

Services Provided by the Link Layer

  • Framing: The encapsulation of Network Layer datagrams into Link Layer frames
  • Link Access: Medium Access Control (MAC) protocol that coordinates frame transmission across multiple nodes
  • Reliable delivery: The Link Layer guarantees that each network-layer datagram will move with out error
    •  Q: If this safeguard is in place why does the Network layer place so much emphasis on ensuring packet integrity?
  • Error detection and Correction: The hardware detects and attempts to fix errors

Error-detection and -correction

  • Parity checks: This is the simplest form of error detection and uses an extra parity bit.

    • The parity bit tells if the number of 1's in the data bits is even or odd.
    EX: The last bit tells us that there is an even number of 1's in the message 1010. 10101

  • 2D-parity checks: The same as a parity check but now the message is represented in matrix so that if multiple errors occur in the same message they can all be found, rather than cancel each other out.

    • A major problem is the limited number of errors that can be found per message, and that the errors are only found and not fixed.

  • Checksumming methods: The same kind of error checking used in the Network Layer
  • Cycle Redundancy Check (CRC): How they work-
    1. Both sender and receiver agree on a pattern of bits R+1 in length, called a generator G
      • The most significant bit (leftmost) must be a 1
      • The generator is labeled "Divisor" in the image below
    2. A packet of Data D is sent with R additional bits, so that the combination of D and R is divisible completely by G
    3. If (D+R)%G != 0 the an error must have occurred
