Q1. What Internet Protocol mechanism is utilized to allow traceroute to function?
A1. The Time To Live mechanism.

Q2. List two methods of TCP Pipelining.
A2. Go-Back-N and Selective Repeat.

Q3. What is the difference between a hub and a switch?
A3. A switch sends data only to the data's specific destination, a hub sends data to everything.

Q4. Name 3 well-known ports and what services use them.
A4. 80: HTTP; 443: HTTP+TLS; 22: SSH; 21: FTP; 23: Telnet; 25: SMTP; 53: DNS;

Q5. What is NAT? Why is it used?
A5. NAT, or Network Address Translation, is a system that allows an entire LAN to be represented by a single external IP address. There is a global shortage of IP addresses.

Q6. What is a Finite State Machine?
A6. An abstract machine that can be in a finite number of states.

Q7. What type of congestion control does TCP Reno use?
A7. Additive-Increase Multiplicative-Decrease (AIMD).

Q8. Explain the relationship between the Maximum Segment Size and the Maximum Transmission Unit.
A8. The Maximum Segment Size is derived from the Maximum Transmission Unit.

Q9. What is the TCP 3-way handshake?
A9. The client sends a Synchronize (SYN) request to the server, the server sends an Acknowledgement (ACK) and a SYN to the client, and the client replies with a final ACK.

Q10. What is the solution to the global IP shortage? How many bits will be in an IP address, and how many IP addresses will be available?
A10. IPv6; 128 bits per address, which means 3.4x10^38 available IP addresses.
