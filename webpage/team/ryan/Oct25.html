Akash: link-state

  • Global routing algorithm
  • Requires a link-state boradcast to gather all infomration about a network
  • advantage is that it is striaght forward, and easy to implement
  • Dijsktra's algorithm: Used to find shortest path between nodes in a graph. Big O of N2
  • Start at a node, check distance to neighbors.
  • Choose node with shorest distance and check distance to it's neighbors
  • Continue until all nodes (that are connected) are checked
  • Any unconnected nodes (or nodes not yet checked who have not been reach by edges so far) are set to infinity
  • Typically repeat for each node so they have a table of how to reach each node
  • Map quest uses Dijsktra's algorithm to calculate route
