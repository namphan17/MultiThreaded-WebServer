  • Javascript
    • Browser-based (originally)
    • High-level, not machine specific, as long as there is an interpreter it will run
    • Interpreted language, not compiled
    • Untyped: You don't need to declare variable types, just declare it as a variable. (If you don't declaire it it'll be global)
    • Dynamic: Can be loaded on the fly, as requested. Not all together at once
    • Functional: Can pass functions as arguments, and have functions return functions as variables/results
    • Netscape 2.0 in 1995 was first ever javascript implementation
    • Internet Explorer 3 in 1996 reversed engineered javascript, arose compatibility issues
    • EMCA in 1997 tried to create a stardard attempt (first attempt), wasn't picked up
    • Ajax was implemented in Javascript in 2005
    • Chrome V8 engine was introduced in 2008 (fast Javascript implementation engine)
    • EMCA finally got a starndard (5.0) in 2009. Now on EMCA 2016
  • V8 engine
    • Made by google
    • Open source
    • Very fast
    • Standalone or embedded system
    • in C++
  • Node.js
    • Created by Ryan Dahl in 2009
    • Based on Google's V8 javascript engine
    • Event-driven, handles things as events occur (listens for even and then does something)
    • Asynchronous, processes things as it happens
    • Good for heavy I/O processing
  • npm
    • Node Packet Manager
    • Full of open source javascript libraries
    • Largest code repository in the world
    • Anyone can publish a package
  • Socket.io
    • Realtime communication engine
    • Uses WebSockets with an AJAX fallback
    • Event-driven (like Node.js)
  • package.json: every node.js file has this. Expresses libraries, name, version, and description.
  • JSON: JavaScript Object Notation, everything is an object, object denoted by brackets. Usually key-value notation. Can also have arrays, denoted by square brackets.
  • server.js is the main driver file. Only thing that it really uses
  • npm install --save____________________command line to generate the folder for running
