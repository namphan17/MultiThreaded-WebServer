Section 5 Notes

  1. Link Layer Implementation

    • The link layer implementation includes certain characteristics as the other layers. For instance, it provides reliable data transfer just like the transport and IP layers.

    • The link layer also incorporates framing, in which the network layer is incorporated in a link layer frame before transmission.

    • The link layer provides certain link access, that is, it's able to transmit from point-to-point (that is, host to node or node to node) via a MAC address (Medium Access Control) which is defined in host's or node's hardware.

    • Lastly, the link layer not only provides error detection (part of the reliable data transfer), but it also performs data correction in many different ways which will be discussed in the next section.

    • Link layer is made in the software of the computer and transmitted via a physical layer (Ethernet, coax cable, fiber, wifi or wireless).
