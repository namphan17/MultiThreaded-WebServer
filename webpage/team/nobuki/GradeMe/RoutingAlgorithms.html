Routing Algorithms

  • The purpose of routing is to find a good path to get from point A to point B
  • Default router/First-hop router: The only router a host is connected to
  • Source router/Destination router: The default router of a source/destination
  • Routing from a source host to a destination host is about routing the source router to the destination router.
  • A good path usually is the path with the least cost.
    • However, there are “real-world concerns” such as policy issues that prevent requests from certain companies be denied on other company routers.
  • The cost of an edge typically may reflect the physical length, the link speed, or monetary cost. The textbook will just give the cost, so don’t worry about it.
  • The cost of edge (x,y) is represented by c(x, y)
  • Globalized Routing Algorithm: Each node has information about distance between all nodes.
    • Link-State algorithm
  • Decentralized Routing Algorithm:Each node only has knowledge on the distance between its neighboring nodes.
    • Distance Vectoring algorithm
