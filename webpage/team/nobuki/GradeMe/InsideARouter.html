What's Inside a Router

  • Forwarding function: the transfer of packets from a router's incoming links to the appropriate outging links at that router.
  • There are four router components.
    • Input Ports
      • It has several key functions.
      • It terminates an incoming physical link to a router.
      • It does functions that are needed to interoperate with the link layer at the other side of incoming links
      • It consults the forwarding table to determine the output port that an arriving packet will be forwarded to
    • Switching Fabric
      • This connects the input ports on the router to the output ports.
    • Output Ports
      • An output port performs necessary link-layer and physical-layer functions to store packets received from the switching fabric and transmits them to the outging link
      • If it is bidirectional, it will be paired with an input port on the same linke card.
    • Routing Processor
      • This does the hard stuff.
      • It executes the routing protocols and performs network management functions
      • It also keeps routing tables and linked state information up to date.
      • Furthermore, it computes the forwarding table for the router.
  • Router forwarding plane: a collective word for forwarding functions that are almost always implemented in hardware.
  • Router control plane: executes the routing protocol, responds to attatched links and perform management functions.
